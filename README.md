# E-commerce Website (React + JSX + Tailwind)

A modern, responsive **E-commerce website** built with **React (JSX)** and **Tailwind CSS**. Browse products, add to cart, and manage your wishlist.

---

## Features
- Product listing with images, titles, and prices  
- Add to Cart & Wishlist  
- Responsive design for desktop & mobile  
- LocalStorage persistence  

---

## Tech Stack
- React (JSX)  
- Tailwind CSS  
- JavaScript  
- Vite
